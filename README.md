# Sketch Notes App

A sketch-style developer notes app to save commands, reminders, and technical notes.

## Features
- Create notes
- Search notes
- Tag-based categories
- Sketch-inspired UI
- Responsive layout

## Tech Stack
- React
- TypeScript
- Vite
- CSS
- Docker

## Run locally
```bash
npm install
npm run dev
```
## Run with Docker
```bash
docker compose up --build
```
